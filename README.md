# ⛷️ Monty Hall Ski Resort Monte Carlo Simulation

This project is an interactive single-page web application that demonstrates the **Monty Hall problem** using a **Monte Carlo simulation**, reimagined with a **winter ski resort theme**.

The app visualizes how probability converges over time and why **switching choices leads to better outcomes**.

---

## 📌 Overview

In this simulation, three ski lift gates represent the Monty Hall choices:
- One gate leads to a **powder run** (win)
- Two gates lead to **icy runs** (loss)

After an initial selection, ski patrol reveals one icy gate. The simulation then compares:
- **Staying** with the original gate
- **Switching** to the remaining gate

---

## ⚙️ Features

- Monte Carlo simulation with **10,000 auto-running trials**
- **Live-updating chart** showing win-rate convergence
- Adjustable simulation speed
- Pause, resume, and reset controls
- Real-time statistics and progress tracking
- Visual reference lines at theoretical probabilities (33.3% vs 66.7%)
- Ski resort UI with falling snow and lift gates
- Built using **HTML, CSS, and vanilla JavaScript**
- Developed with assistance from **ChatGPT and Claude**

---

## 📈 Expected Results

As the number of trials increases:
- **Switching** converges to approximately **66.7%**
- **Staying** converges to approximately **33.3%**

This demonstrates the **Law of Large Numbers** and validates the theoretical solution to the Monty Hall problem.
